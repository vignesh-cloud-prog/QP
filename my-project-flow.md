Install python

* Download python virtualenv module 
  ```shell
  pip install virtualenv
  ```
* Create virtual environment
  ```shell
  virtualenv QP_env
  ```
* Activating the virtual environment
  ```shell
  .\QP_env\Scripts\activate
  ```
* Installing Django
  ```shell
  pip install Django
  ```
* Create QP Web Project
  ```shell
  django-admin startproject qp
  ```
